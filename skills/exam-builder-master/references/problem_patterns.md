# 국어/영어 문제 출제 패턴 및 오답 설계 가이드

이 문서는 에이전트가 문제를 생성할 때 참고해야 할 기준입니다.

## 1. 매력적인 오답 (Distractor) 설계 원칙
- **부분적 참 (Partial Truth)**: 지문에 언급된 단어나 내용을 포함하지만, 전체적인 논리나 주장이 빈칸/질문과 맞지 않는 오답.
- **반대 논리 (Opposite Logic)**: 지문의 핵심 내용과 정반대되는 내용을 담은 오답. (특히 역접 접속어 문제에서 유용)
- **과도한 일반화 (Overgeneralization)**: 지문의 특정 사례를 전체로 지나치게 확대 해석한 오답.

## 2. 빈칸 추론 (Blank Inference) 패턴
- **목표**: 글의 핵심 주장이나 결정적인 근거가 되는 부분을 빈칸으로 뚫어, 전후 문맥을 통해 논리적 사고를 묻는다.
- **가이드**: 빈칸 주변의 '접속어(그러나, 예를 들어 등)'나 '지시어(이러한, 그 등)'가 강력한 힌트가 되도록 지문을 정제한다.

## 4. 선지(Options)의 복잡도 및 길이 강화 패턴

단답형 선지는 변별력이 낮으므로, 다음과 같이 **지문의 논리 구조가 포함된 긴 문장**으로 선지를 구성합니다.

| 유형 | [단순/짧은 선지] (지양) | [수능형/고품질 긴 선지] (지향) |
| :--- | :--- | :--- |
| **주제** | AI 기술의 발전 속도 | AI 기술의 비약적 발전에 따른 정보 처리 과정의 유사성과 그에 따른 자아 의식의 창발 가능성에 대한 철학적 고찰 |
| **일치** | 데이터 학습은 차별을 낳는다. | 방대한 데이터 학습 과정에서 데이터 자체에 내재된 편향성이 증폭됨으로써, AI가 편파적인 결과를 도출하여 특정 집단을 차별할 가능성이 높다. |
| **추론** | 인간의 판단은 정확하다. | AI가 통계적 확률에 기반하여 결과물을 도출하는 한, 인간이 가진 고유한 주관적 경험인 퀄리아(Qualia)를 완벽히 재현하기는 어려울 것으로 추론된다. |

- **가이드**: 선지 내에 '원인-결과', '조건-결과'와 같은 논리적 연결어를 포함시켜 수험생이 지문의 맥락을 깊이 파악해야만 풀 수 있도록 구성하세요.
