# 전문 인쇄를 위한 PDF 디자인 표준 가이드 (Print Standards)

본 가이드는 AI 에이전트가 시험지를 발행할 때 '인쇄소 즉시 발주 가능' 수준의 품질을 확보하기 위한 기준입니다.

## 1. 페이지 레이아웃 및 여백 (Layout & Margins)
- **도련 (Bleed)**: 사방 3mm 추가 (최종 결과물이 종이 끝까지 인쇄될 경우 필수).
- **재단선 (Crop Marks)**: 인쇄 후 자르는 위치를 표시하는 선을 포함.
- **제본 여백 (Gutter)**: 책자 형태 제본 시 안쪽(Inside) 여백을 바깥쪽보다 5~10mm 더 넓게 설정.
- **안전 영역 (Safe Zone)**: 재단선 안쪽으로 5mm 이내에는 중요한 텍스트나 로고를 배치하지 않음.

## 2. 색상 및 이미지 (Color & Assets)
- **색상 공간 (Color Space)**: 반드시 CMYK 프로파일 사용 (Typst 컴파일 시 설정 확인).
- **해상도 (Resolution)**: 삽입되는 모든 이미지(로고, 도표)는 최소 300 DPI (벡터 방식인 SVG 권장).
- **K100 (Rich Black)**: 검은색 텍스트는 C:0 M:0 Y:0 K:100으로 설정하여 인쇄 시 번짐 방지.

## 3. 타이포그래피 (Typography)
- **폰트 임베딩**: PDF 생성 시 모든 폰트 데이터가 포함되도록 설정.
- **최소 폰트 크기**: 본문 9.5pt 이상, 각주 및 설명 7pt 이상 유지.
- **가독성**: 국어는 명조/고딕 계열, 영어는 Serif(시인성) 계열 폰트 혼용.

## 4. Typst 구현 팁
- `examify` 패키지나 `g-exam` 템플릿의 로직을 참고하여 문항 번호와 점수를 자동 계산함.
